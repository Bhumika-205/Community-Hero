# CommunityHero 

CommunityHero is an AI-powered civic issue reporting platform that helps citizens report, track, and verify local community problems such as potholes, sanitation issues, water shortages, and public safety concerns.

## Features

-  Interactive live map for issue reporting
-  AI-powered issue classification using Google Gemini
-  Severity and priority prediction
-  Community verification and voting system
-  Impact analytics dashboard
-  Citizen engagement and rewards system
-  Cloud-hosted backend with MongoDB Atlas

## Tech Stack

**Frontend**
- React.js
- Leaflet Maps
- Tailwind CSS

**Backend**
- Node.js
- Express.js
- MongoDB Atlas

**AI & Cloud**
- Google Gemini API
- Cloudinary
- Google Cloud

## Problem Solved

CommunityHero enables citizens and local authorities to collaborate efficiently by providing a transparent, AI-assisted platform for reporting and prioritizing community issues.
